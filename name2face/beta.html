Name2Face

Name2Face is a tool to help you learn new faces.

  • You can use Name2Face to create a set of faces and names to quiz yourself with.
  • Name2Face keeps track of your success and displays the faces you have the most difficulty identifying.
  • Name2Face strengthens retention by requiring you to type the name in.
  • Name2Face was originally designed to be used by teachers to learn the names of students in their class.
