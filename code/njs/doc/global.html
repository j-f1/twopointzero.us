(nullable) Action()

A callback for a button on a notification. Possible values:

  • function(event): passed the same arguments as the callback to $.fn.click. If the return value is falsy but not undefined, the notification will not dismiss.
  • Boolean: This is equivalent to specifying a function that always returns this value.
  • undefined or null: Equivalent to true.
